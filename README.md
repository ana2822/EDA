# EDA

# Task 2: Exploratory Data Analysis (EDA)

## Objective

To analyze and understand the dataset using statistical and visualization techniques.

## Dataset

Titanic Dataset containing passenger details such as age, fare, class, and survival status.

## Steps Performed

* Loaded dataset using Pandas
* Performed data understanding (info, describe, missing values)
* Generated summary statistics (mean, median, standard deviation)
* Created histograms to understand distribution
* Used boxplots to detect outliers
* Built correlation heatmap to analyze relationships
* Used pairplot for feature interactions
* Performed additional analysis using countplots and groupby

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Key Insights

* Most passengers were young adults
* Fare column contains outliers
* Female passengers had higher survival rate
* Higher class passengers survived more
* Weak correlation between most features

## Conclusion

EDA helped in understanding the dataset, identifying patterns, detecting outliers, and analyzing relationships between variables.
